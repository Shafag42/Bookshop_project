CSS yəni Cascading Style Sheet dilimizə "Kaskadlı üslub cədvəlləri" kimi tərcümə olunur.CSS veb səhifələrimizə müəyyən görünüş,üslublar verərək onlara vizualllıq verir. Bu xüsusiyyətlərə rəng,font,size,arxa fon ölçülərini göstərə bilərik. CSS`dəki bir çox funksiyanı əslində Html`də də edə bilsək də işlərimizin daha asan və sürətli olması cəhətdən CSS böyük kömək edir. Beləliklə CSS veb səhifələrin kodlaşdırılmasında istifadə olunan üç ən əsas veb texnologiyalar arasındadır.
Həmçinin CSS ilə biz veb səhifəyə asanlıqla  maraqlı animasiyalar,gif`lər, səs yazıları və s.əlavə edə bilərik. Beləliklə də CSS ilə biz vaxta qənaət edərək veb səhifələrimizi gözəlləşdirə, müxtəlif ölçülü ekranlar üçün(telefon,planşet,notbuk və s.) onları uyğunlaşdıra bilirik.Və ən əası da bir CSS faylı bir neçə HTML sənədi üçün də istiadə edə bilərik.
Selectorklar ilə CSS`də stil vermək istədiyimiz HTML elementlərini tapmaq üçün istifadə olunur. Bunlara bizim hazırda ən çox istifadə etdiyimiz sadə id,class,name selekorlarını göstərə bilərik.Ümumilikdə CSS selektorlar 5 kateqoriyaya bölünür.
Bütün HTML  elementləri seçmək üçün "*" selektorundan istifadə edirik. "#" işarəsi ilə id`ləri ,"."ilə classları tapmaq üçün istifadə olunur.Və id,class adları rəqəmlər ilə başlamamalıdır.
Qrup selektorlarına isə ən hazırda ən çox istifadə etdiyimiz h1,h2,..,p yaza bilərik.Bunları CSS`də yazdığımız zaman xüsusi işarələrdən istifadə etmirik eləcə adlarını yazıb sonra bəzəkli mörtərizələr ilə digər əməliyyatları davam etdiririk.
